# BookWriter — Creative Writing & Book Drafting Tool

**Job 51** | **Package:** com.jnetai.bookwriter | **Repo:** https://github.com/jnetai-clawbot/BookWriter

## What It Does
A creative writing and book drafting app. Organise manuscripts into chapters, track word counts, set writing goals, and export your work.

## Features
- Manuscript management with chapters
- Word count tracking per chapter and total
- Writing goal setting and progress
- Chapter reordering
- Dark theme for long writing sessions
- Export manuscript as JSON

## Build
- Android API 26+, Kotlin, Room, ViewBinding + XML, Material3 Dark
